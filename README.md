# Note App – Mobile Devices Programming Project

## 📱 Project Overview
This application was developed as part of the university exam **"Mobile Devices Programming"**.

It represents the **first mobile application** I have ever created, using **Android Studio** as the main development environment.

The goal of the project was to explore the fundamentals of Android development, including UI design, file management, and interaction features.

---

## 🚀 Features

- ✍️ **Text Saving**
  - Users can write text inside the central input field.
  - The text can be saved locally in the device directory.

- 📂 **Saved Text Display**
  - By clicking the dedicated button, the app retrieves the saved text.
  - The content is shown on screen using a **Toast message**.

- 🔊 **Text-to-Speech**
  - By clicking on the ear icon, the app reads aloud the text written in the input field.

- 🎨 **Drawing Board**
  - By clicking on the pen icon, a drawing canvas opens.
  - Users can draw freely on the screen.
